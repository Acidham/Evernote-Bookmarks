# Evernote Bookmark
Find URLs in Evernote notes titles (inline) tagged with a URL Tag. The URL tag can be changed in Workflow Preferences.

## Usage:
* ***enb*** \<*query>*
- `SHIFT` for quicklook url
- `CMD` to copy to clipboard

## Requires
* Evernote installed
